# Digital Marketing Project Management — ClickUp

 A fully structured project management system built in ClickUp to manage end-to-end digital marketing campaigns for clients.

# Project Overview

This workspace was designed to manage a **complete digital marketing campaign lifecycle**, from pre-launch preparation to campaign execution. It was built for a client's digital marketing team to ensure every stakeholder, asset, and task was tracked in one centralized place.

# Tools Used

- **ClickUp** — Task and project management
- **List & Board Views** — For flexible team workflow visualization
- **Custom Status Tags** — For clear phase tracking
- **Subtasks & Sub-phases** — For granular task breakdown

# Project Structure

The project was broken into **5 key phases:**

 **PREP**: Team roster setup, stakeholder interviews, tracking meetings, asset collection 

 **PLAN**: Campaign strategy, content planning, timeline mapping 

 **BUILD**: Asset creation, account setups (GA4, Google Ads, YouTube, etc.) 

 **IMPLEMENT**: Campaign execution and deployment 

 **LAUNCH CAMPAIGN**: Go-live, monitoring, and reporting 

# Key Features of This Setup

- **Team Roster & Access Management** — Defined roles and permissions for all team members
- **Stakeholder Interview Tracking** — Documented and tracked client interviews
- **Asset Logs** — Separate logs for Video Assets, Display Assets, and Landing Pages
- **Account Setup Tracking** — GA4, Google Ads, YouTube, and other platform setups tracked as tasks
- **Sub-phase Labeling** — Each task tagged by sub-phase (Project Prep, Client Assets, Account Setup) for easy filtering
- **Due Date Monitoring** — All tasks assigned deadlines for accountability

# Problems This Solved

- Eliminated confusion about **who is responsible for what**
- Gave the client **real-time visibility** into campaign progress
- Ensured **no asset or account setup was missed** before launch
- Created a **repeatable template** for future campaigns
